МІЙ ЕЛЕКТРОННИЙ ДОВІДНИК HTML
Призначення Назва тегу Короткий опис Стандарт
Для каркасу документа <html> Є контейнером, який містить в собі весь вміст веб-сторінки HTML5
<head> Призначений для зберігання елементів, мета яких - допомогти браузеру в роботі з даними HTML5
<body> Призначений для зберігання змісту веб-сторінки, що відображається у вікні браузера HTML5
Для заголовка документа <title> Використовується для відображення рядка тексту в лівому верхньому кутку вікна браузера, а також на вкладці HTML5
<meta> Використовується для зберігання інформації, призначеної для браузерів і пошукових систем HTML5
Для роботи з текстом <p> Визначає текстовий абзац HTML5
<br> Встановлює новий рядок в тому місці, де цей тег знаходиться HTML5
<pre> Визначає блок попередньо форматованого тексту, який відображається з усіма пробілами між словами. HTML5
<b> Виділяє текст жирним шрифтом HTML5
<strong> Призначений для акцентування тексту,раузери відображають такий текст жирним шрифтом HTML5
<i> Виділяє текст курсивом HTML5
<em> Призначений для акцентування тексту, браузери відображають такий текст курсивом HTML5
<u> Додає підкреслення до тексту HTML5
<s> Відображає текст як перекреслений HTML5
<h1>...<h6> Заголовок 1...6 рівнів HTML5
Для гіперпосилань <a> Призначений для створення посилань HTML5
Для зображень <img> Призначений для відображення на веб-сторінці зображень в графічному форматі HTML5
Для роботи з таблицями <table> Служить контейнером для елементів, що визначають вміст таблиці HTML5
<th> Призначений для створення однієї комірки таблиці, яка позначається як заголовна HTML5
<tr> Служить контейнером для створення рядка таблиці HTML5
<td> Призначений для створення однієї комірки таблиці HTML5
Для роботи з списками <li> Визначає окремий елемент списку HTML5
<ol> Встановлює нумерований список, кожен елемент списку починається з числа або літери і збільшується по наростаючій HTML5
<ul> Встановлює маркований список, кожен елемент якого починається з невеликого символу - маркера HTML5
